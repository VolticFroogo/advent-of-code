# My [Advent of Code](https://adventofcode.com/) Solutions
My solutions to [Advent of Code](https://adventofcode.com/) written in Go, just going through 2025 for now

Not all solutions are optimal, though I avoid pure brute-forcing

### How to run
1. [Install Go](https://go.dev/doc/install)
1. Create `input.txt` file in relevant day folder
1. Open terminal in relevant day folder (must be correct directory for reading input file)
1. Run `go run part-x.go` where `x` is the part number
